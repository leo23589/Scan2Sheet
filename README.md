# Scan2Sheet
Scan2Sheet est une application permettant d'automatiser la saisie des dépenses. L'utilisateur prend en photo un ticket de caisse, et l'application se charge d'extraire intelligemment les informations clés (Montant total, Date, Marchand, TVA) grâce à la reconnaissance optique de caractères (OCR) pour les envoyer automatiquement dans un tableur
