# 🧾 [Scan2Sheet]

Une solution de bout en bout pour transformer les photos de vos tickets de caisse en données structurées et exploitables. 

Ce projet capture l'image d'un reçu, utilise la reconnaissance optique de caractères (OCR) pour en extraire les informations clés (Date, Montant total, Marchand, Catégorie) et synchronise automatiquement ces données vers un tableur externe.

## ✨ Fonctionnalités

* **Capture d'image :** Interface permettant l'upload ou la prise de photo directe d'un reçu.
* **Extraction intelligente (OCR) :** Détection et isolation automatiques des champs cruciaux du ticket.
* **Synchronisation tableur :** Ajout d'une nouvelle ligne formatée dans le document cible à chaque validation de scan.
